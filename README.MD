🎮 Игромир (ToyVerse)Геймифицированная система лояльности для сетей детских магазинов.«Игромир» — это инновационное мобильное приложение, которое стирает грань между физическими покупками в магазине игрушек и увлекательным цифровым миром. Приложение превращает обычную программу лояльности (скидки и бонусы) в полноценную RPG-игру и энциклопедию для детей, одновременно предоставляя выгоду родителям.🌟 Концепция проектаОсновная проблема классических программ лояльности — дети в них не вовлечены, а родители часто забывают пластиковые карты. «Игромир» решает это через игровой цикл (Core Loop):Изучение и Вовлечение: Ребенок листает каталог-энциклопедию, где каждая физическая игрушка представлена как цифровой артефакт с RPG-характеристиками (здоровье, магия, стихия, лор).Покупка: Родитель покупает игрушку в физическом магазине. На накопленный в приложении уровень дается реальная скидка.Магия на кассе: Продавец генерирует уникальный QR-код на своем устройстве. Ребенок сканирует его через приложение.Награда: Купленная игрушка навсегда разблокируется в виртуальной «Коллекции» профиля, персонаж получает опыт (XP), энергию и повышение уровня.Игра: Открытые игрушки используются во встроенном игровом модуле (ToyVerse) для прохождения уровней и новых приключений.🚀 Основной функционал👦 Для игроков (Дети и Родители)Ролевой профиль: Отображение уровня героя, баланса бонусов и накопленной скидки.Энциклопедия артефактов: Умный каталог с фильтрами по коллекциям (Minecraft, Драконы и т.д.), редкости и стихиям. Динамический расчет цены с учетом скидки за уровень.RPG-карточки: Подробные характеристики каждой игрушки (Сила, Ловкость, Интеллект) и её лор (история, слабости, среда обитания).Виртуальная коллекция: Разблокировка силуэтов игрушек после реальной покупки.Встроенный сканер: Чтение одноразовых QR-кодов для зачисления лута и бонусов.Игровой модуль: Бесшовный переход во встроенный игровой мир прямо из профиля.🛡️ Для продавцов (Администраторы)Мобильная панель управления: Встроенный прямо в приложение скрытый интерфейс.Управление каталогом: Добавление и редактирование игрушек с телефона. Загрузка фото из галереи с автоматическим сжатием и конвертацией в Base64.Генерация QR-кодов: Создание одноразовых безопасных чеков (токенов) для сканирования покупателем прямо на кассе.👑 Для руководства (Суперадмины)Управление персоналом: Выдача и отзыв прав Администратора (продавца) у обычных пользователей в пару кликов.🛠 Стек технологийFrontend (Мобильное приложение):React NativeExpo (Camera, Image Picker, Image Manipulator)React Native Safe Area ContextНавигация: Кастомная стейт-навигация для максимальной производительности и бесшовной интеграции игрового модуля.Backend (API сервер):Python 3FastAPI (Высокопроизводительный асинхронный веб-фреймворк)SQLAlchemy (ORM для работы с базой данных)SQLite (Легковесная база данных)Pydantic (Валидация данных)⚙️ Установка и запуск (Для разработчиков)1. Запуск Backend-сервераУбедитесь, что у вас установлен Python (3.8+).Перейдите в папку с бэкендом.Установите зависимости:pip install fastapi uvicorn sqlalchemy pydantic
Запустите сервер (база данных database.db создастся автоматически при первом запуске):python -m uvicorn main:app --host 0.0.0.0 --reload
2. Настройка Frontend-приложенияУбедитесь, что у вас установлен Node.js.Узнайте ваш локальный IPv4 адрес (команда ipconfig для Windows или ifconfig для Mac/Linux).Откройте файл src/api/config.js и впишите туда ваш IP-адрес с портом 8000:export const API_URL = 'http://ВАШ_IP_АДРЕС:8000';
Установите зависимости проекта:npm install
# или используйте npx expo install для специфичных библиотек
npx expo install expo-camera expo-image-picker expo-image-manipulator @react-native-picker/picker react-native-safe-area-context
Запустите проект с очисткой кэша:npx expo start -c
Отсканируйте появившийся QR-код камерой телефона (или приложением Expo Go), чтобы открыть приложение.🎮 Инструкция по тестированиюПолучение прав Супер-Администратора: При первой регистрации создайте пользователя с логином chief. Сервер автоматически распознает это имя и выдаст вам максимальные права.Управление ролями:Зайдите под chief во вкладку "Доступ" и назначьте другим зарегистрированным аккаунтам роль "Админ".Добавление товаров:Зайдите под аккаунтом Админа во вкладку "Управление", нажмите "+ Создать героя", заполните RPG-характеристики и добавьте фото из галереи.Тестирование покупки:Как Админ: нажмите "Изменить" на товаре -> сгенерируйте QR-код.Как Клиент: откройте вкладку "Сканер", наведите камеру на QR-код Админа. Зайдите в Профиль и проверьте пополнение виртуальной коллекции и баланса!🗂 Структура проекта/src/api — Конфигурация API (IP сервера)./src/components — Переиспользуемые компоненты (например, верхняя навигация TopNav)./src/screens — Основные экраны приложения (Каталог, Профиль, Админка, Сканер)./src/game — Изолированный модуль встроенной игры (карта, уровни).main.py — Точка входа FastAPI бэкенда.models.py / schemas.py — Структура базы данных и валидация.Разработано с душой для будущих героев! 🐉
